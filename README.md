# Şahmeran Yapı - Modern Construction Company Website

[![Live Demo](https://img.shields.io/badge/Live%20Demo-View-%23308d46?style=for-the-badge&logo=vercel&logoColor=white)](https://sahmeranyapi.netlify.app/) 

Şahmeran Yapı is a modern and professional corporate website that elegantly presents company services, completed projects, testimonials, and contact information to visitors.

Built as a fully static site, it is fast, SEO-friendly, and highly performant. The responsive design, crafted with Tailwind CSS, ensures a flawless experience across mobile, tablet, and desktop devices.

## Features

- **Modern and Clean Design**: Minimalist aesthetic with eye-catching visuals and subtle animations
- **Fully Responsive**: Perfectly optimized for all screen sizes and devices
- **Dynamic Content Loading**: All sections (header, hero, services, projects, etc.) are loaded asynchronously using vanilla JavaScript
- **Project Gallery**: Category-based filtering powered by MixItUp and lightbox viewing with GLightbox
- **Testimonials Slider**: Smooth, auto-playing customer reviews built with Swiper.js
- **Smooth Scrolling & Active Menu**: Header background changes on scroll, and the current section is highlighted in the navigation
- **Mobile Menu**: Intuitive hamburger menu for seamless mobile navigation
- **Contact Form**: Easy-to-use form for visitor inquiries (ready for backend integration)
- **Performance-Oriented**: Utilizes GSAP ScrollTrigger, lightweight CDN libraries, and optimized code structure

## Technologies Used

| Category         | Technology                  | Description                                      |
|------------------|-----------------------------|--------------------------------------------------|
| **Structure**    | HTML5, Vanilla JavaScript   | Lightweight and high-performance static site     |
| **Styling**      | Tailwind CSS                | Utility-first framework for rapid and consistent design |
| **Slider**       | Swiper.js                   | Touch-friendly, modern carousel slider           |
| **Lightbox**     | GLightbox                   | Elegant and fast image lightbox experience       |
| **Gallery Filter**| MixItUp                     | Smooth category-based project filtering          |

## Installation and Local Running

The project is completely static and requires no build tools, dependencies, or backend.

### Requirements
- A modern web browser
- (Optional) Local server tool such as Live Server (VS Code extension)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/kadirsahmeran/sahmeran-yapi-landing-page.git
   cd sahmeran-yapi-landing-page
2. Preview the project locally:
   - **Recommended (VS Code users):** Open the project folder in Visual Studio Code and click **"Go Live"** (using the Live Server extension). This starts a local server and provides the best experience.
   - **Alternative:** Simply open the `index.html` file directly in your web browser.
